```Final grade = 0.7*HW + 0.3*Exam```

* `Exam` -  Grade for the exam
 
 ---
 
You can skip the exam if your average grade for the first 5 assignemnts is **not smaller** than 6 (`HW >=6`).

```Final grade = HW```
